Create Variables with Chunk.

Chunky has it's own built in variable storage and documentation system. For now, let's create a variable. Use chunk.data("Variable Name") to create a variable. Use chunk.data("Variable Name", Any Value Here) to set the value. Add .getValue() to it to retrieve the value. Variables in a chunk are referred to as "data". Data written to the Chunk will appear under the "dataTree" property for this Chunk.
